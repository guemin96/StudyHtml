## 1. 기본 선택자 
- 기본선택자 종류
  - 전체 선택자 (\*\) : HTML 페이지 내부의 태그를 모두 선택
  - 태그 선택자 (태그) :HTML 페이지 내부의 특정 태그를 모두 선택
  - 아이디 선택자 (#아이디) : 특정 id 속성이 있는 태그 선택, 웹 표준에 id 속성은 웹 페이지내부에서 중복되면 안 됨.그래서 특정 태그 하나를 선택할 때 사용
  - 클래스 선택자 (.클래스) : 특정 클래스가 있는 태그
